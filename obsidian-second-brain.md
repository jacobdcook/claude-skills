# Obsidian second brain

This skill is maintained as a **bundle** (skill + 12 command stubs):

**→ [obsidian-second-brain/SKILL.md](obsidian-second-brain/SKILL.md)**  
**→ [obsidian-second-brain/commands/](obsidian-second-brain/commands/)**

Quick install: copy `obsidian-second-brain/SKILL.md` to `~/.claude/skills/obsidian-second-brain/SKILL.md`, copy each file from `obsidian-second-brain/commands/` into `~/.claude/commands/` (or `~/.cursor/commands/`), then replace **`YOUR_OBSIDIAN_VAULT`** with your vault absolute path.

See **`obsidian-second-brain/README.md`** in the same folder for a one-line summary.
