# Obsidian second brain (bundle)

Vin-style vault slash commands for Claude Code / Cursor + a primary Obsidian vault.

- **Start here:** [`SKILL.md`](SKILL.md)  
- **Slash helpers:** [`commands/`](commands/) (12 files). Copy into `~/.claude/commands/` or `~/.cursor/commands/`, then replace **`YOUR_OBSIDIAN_VAULT`** with your vault path.

Upstream lives in this repo for sharing; fork and adjust **`CLAUDE.md`** inside your vault to match your schema.
