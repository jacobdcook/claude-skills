# /challenge — Pressure-test beliefs using vault history

**Vault:** `YOUR_OBSIDIAN_VAULT` — **`CLAUDE.md`**.

Pressure-tests **current beliefs** using the vault’s **own history**: contradictions, **counter-evidence**, and **shifts in thinking** over time.

**Topic** from message after **`/challenge`**. Ask once if empty.

## Do this

1. Gather **`rg`** hits in **`wiki/`** (and **`raw/`** read-only if needed); note **dated** or **sequential** contradictions, not only static tension.
2. List **tensions**: contradictions, unstated assumptions, single-source claims, obsolete dates.
3. For each tension: **quote or paraphrase** with paths; name **counter-evidence** elsewhere in the vault when it exists.
4. Propose **neutral tests** (what evidence would flip the belief). No wiki writes unless user asks.
