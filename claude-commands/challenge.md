# /challenge — Pressure-test beliefs on a topic

**Vault:** `YOUR_OBSIDIAN_VAULT` — **`CLAUDE.md`**.

**Topic** from message after **`/challenge`**. Ask once if empty.

## Do this

1. Gather **`rg`** hits in **`wiki/`** (and **`raw/`** read-only if needed).
2. List **tensions**: contradictions, unstated assumptions, single-source claims, obsolete dates.
3. For each tension: **quote or paraphrase** with paths; propose **neutral tests** (what evidence would flip the belief).
4. Stay constructive. No wiki writes unless user asks.
