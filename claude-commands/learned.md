# /learned — Turn vault material into three drafts

**Vault:** `YOUR_OBSIDIAN_VAULT` — **`CLAUDE.md`**.

Mines the vault for **raw material** on a topic and generates **three writing drafts**: **short post**, **personal essay**, and **universal essay** (increasing abstraction). Topic follows **`/learned`**.

## Do this

1. **`rg`** the topic across **`wiki/`** and (read-only) **`raw/notes/`**; collect 10–25 excerpt-worthy lines with paths.
2. Produce **Draft A** (~200–400 words): punchy post.
3. Produce **Draft B** (~600–900 words): personal narrative with specifics.
4. Produce **Draft C** (~800–1200 words): general-audience essay; strip over-identifying details unless user wants otherwise.
5. Save under **`wiki/`** only if user confirms path + frontmatter per **`CLAUDE.md`**.
