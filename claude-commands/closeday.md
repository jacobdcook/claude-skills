# /closeday — End-of-day capture (counterpart to /today)

**Vault:** `YOUR_OBSIDIAN_VAULT` — **`CLAUDE.md`**.

User adds what they worked on inline if desired.

## Do this

1. If CLI works: read today’s **`obsidian daily:read`**; else open today’s daily file from vault.
2. Summarize **done / learned / blocker / carry-over** bullets (explicitly label uncertainty).
3. Optional append to daily via **`obsidian daily:append`** if CLI OK **and** user wants it persisted.
4. If something belongs in **`wiki/`** (decision level), propose **`wiki/decisions/`** or **`wiki/log.md`** + index update rather than burying only in daily.
