# /contradict — Incompatible beliefs held at once

**Vault:** `YOUR_OBSIDIAN_VAULT` — **`CLAUDE.md`**.

Finds **beliefs held simultaneously** that are **logically incompatible**. This is **not** “you changed your mind over months”; it is **tension right now** across notes.

Optional **`[domain]`** after **`/contradict`** narrows search (e.g. `career`, `health`).

## Do this

1. If domain given: **`rg`** in **`wiki/<domain>/`** and linked dailies; else scan **`wiki/`** + recent **`raw/notes/`** (read-only) for strong normative claims (“always”, “never”, “must”, “cannot”).
2. Cluster claims into pairs that **cannot both be true** under the same premises; quote short snippets with paths.
3. For each pair: state **A**, state **B**, and the **hidden assumption** that forces the clash.
4. Suggest **one resolution path** (reframe, split context, or pick a side) without editing vault until the user approves.
