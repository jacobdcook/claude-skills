# /weekly-learnings — Team email prep from the vault week

**Vault:** `YOUR_OBSIDIAN_VAULT` — **`CLAUDE.md`**.

Prepares material for a **weekly team email**: patterns, insights, and **candidate topics** from roughly the **past seven days** of notes (adjust if user specifies).

## Do this

1. Scan **last 7 daily notes** + **`wiki/log.md`** diff mentally (tail read) + any **`wiki/`** pages touched this week (mtime or git if repo).
2. Extract **3–6 learnings** suitable for a professional audience; scrub private names if needed.
3. Output **Email skeleton**: subject line, **TL;DR** (3 bullets), **body** with sections, **optional deeper links** as internal paths only.
4. Redact or generalize anything too personal unless user says keep raw.
